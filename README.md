# Учет посещаемости

Приложение для учёта посещаемости лиц в учебной организации по расписанию и формирование отчетности по посещаемости.
Возможности приложения: 
- Добавление/редактирование лиц для учета
- Составление расписания помесячно, с указанием стоимости абонемента
- Заполнение посещаемости с разнымими статусами на каждый день из расписания
- Формирование отчета по посещениям в расписании

# Проверено
- MacOS (maccatalyst)
- Windows 11
